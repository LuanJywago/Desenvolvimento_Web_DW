Exemplo 10 Revisão - HTML/CSS - Formas de Vinculação

Vinculação Externa(Importadas ou Linkadas)

Nesse método, o css é escrito em um arquivo separado com extensão .css e, então, ele é vinculado de forma externa no documento html por meio da tag link no head do documento html

Nesta página estamos usando a vinculação externa. Ela é importante, pois trás algumas vantagens:

  • Organização: Mantém o código CSS organizado e separado do HTML.
  • Reutilização: O mesmo arquivo pode ser reutilizado em várias páginas distintas.
  • Performance: O navegador pode armazenar o arquivo CSS em Cache, melhorando o carregamento da página
