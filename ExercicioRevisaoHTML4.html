Exemplo 44 HTML - Elementos HTML (Listas Ordenadas e Não Ordenadas)

Listas

Listas em html são representadas pelas tags <ol> (Lista Ordenada) <ul>(Lista não Ordenada). Essas listas possuem diversas finalidades, popdendo ser utilizadas para exibir informações ou simplesmente serem usadas em menus.

Toda vez que usarmos listas ordenadas ou não ordenadas, devemo utilizar a tag item <li>

Exemplo de Lista Ordenada

  1. Item lista 1
  2. Item lista 2
    1. Subitem 2.1
    2. Subitem 2.2
    3. Subitem 2.3
    4. Subitem 2.4
  3. Item de lista 3

  6. Item de Lista 6
