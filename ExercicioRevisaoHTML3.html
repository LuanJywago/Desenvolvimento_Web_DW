Exemplo 3 HTML - Estrutura básica de um documento HTML

Este exemplo mostra a estrutura minima de um documento html. A seguir, são descritas as principais tags utilizadas

  • < !DOCTYPE html >
    • Define o tipo do documento html
  • < html >É o elemento central, o container, de todo documento hmtl
  • <head >Contém os metadados que serão interpretados pelo navegador, tais como: títulos de páginas, links para arquivos externos e meta arquivos de configuração do documento.
  • <body >Usamos essa tag para deixar visível todo conteúdo da pagina ou documento html

Abaixo está o codigo completo deste exemplo:

            <!DOCTYPE html>
        <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo 1 - Estrutura Básica</title>
          </head>
          <body>
            <h1>Bem-vindos à minha página!</h1>
            <p>Conteúdo principal aqui</p>
            <p>Rodapé da página</p>
          </body>
        </html>
